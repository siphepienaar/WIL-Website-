Objective

To define the hierarchy, responsibilities, and reporting structure of the ICT department managing the campus network.

IT Staff and Responsibilities

  • ICT Officer: Oversees all IT operations and policies.
  • Network Administrator: Manages routers, switches, and wireless access.
  • System Administrator: Maintains servers, backups, and updates.
  • Help Desk Technician: Handles user issues and tickets.
  • Security Specialist: Ensures network and data security.
  • Procurement Officer: Manages ICT purchases and asset register updates.

Reporting Structure

ICT Officer → Network/System/Security Specialists → Help Desk → End Users
