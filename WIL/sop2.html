SOP 2: IT Security Policy

← Back to Home

Objective

To ensure the protection of all ICT systems, data, and infrastructure from unauthorized access or misuse.

Security Principles

  • Use strong passwords and enable multi-factor authentication.
  • Restrict administrative access to authorized personnel only.
  • Regularly update all systems and security patches.
  • Conduct quarterly security audits and risk assessments.

Physical & IT Security

  • Server rooms locked with access control.
  • Install CCTV surveillance at critical points.
  • Ensure all devices have ESET PROTECT Advanced Cloud installed.
