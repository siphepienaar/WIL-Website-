SOP 47: Website SOP

← Back to Home

Objective

To ensure the ICT SOP website remains functional, secure, and regularly updated.

Maintenance Schedule

  • Weekly file and database backups.
  • Quarterly content review by ICT Officer.
  • Annual redesign or content refresh.

Security

  • Access restricted to authorized web admins.
  • Strong password policies enforced.
  • SSL certificates renewed annually.
